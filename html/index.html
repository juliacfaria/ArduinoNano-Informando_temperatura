O que é Arduino

O Arduino é um microcontrolador open-source que foi criado com o objetivo de facilitar a prototipação de projetos, de forma que iniciantes na área da computação possam desenvolver um projeto simples aprendendo o básico desse componente. Além disso, o Arduino tem seu próprio ambiente de programação, assim, ele abrange hardware (parte física) e software (parte lógica). O código é enviado para a placa que, ao receber as informações, as interpreta e realiza seus comandos. Para informações mais detalhadas sobre o Arduino, clique aqui. Por ser uma plataforma favorável ao desenvolvimento de projetos, ser fácil de manusear e ideal para iniciantes, o arduino foi a base do trabalho em questão. Sobre o Projeto

A variação da temperatura é importante em muitas áreas da atualidade, sejam elas relacionadas ao trabalho ou à pesquisa. Essa variação pode influenciar no comportamento de animais, na velocidade de reações químicas, na conservação de alimentos e máquinas, dentre outros. Para facilitar o processo de medição de temperatura, o projeto visou desenvolver um aparelho que realizasse tais medições, além de disponibilizá-las online e em tempo real.

Para isso, utilizou-se o Arduino como base do equipamento e o interligou a outros componentes eletrônicos de forma a realizar todas as funcionalidades definidas no escopo do projeto. A montagem final do projeto está indicada na figura 2.

Os componentes eletrônicos usados no projeto foram: 

    
   
  • uma placa de rede Ethernet Shield W5100;
   
  • um sensor de temperatura a prova d’água DS18B20;
   
  • um display LCD 16x2;
   
  • uma protoboard; 
        
  • jumpers;
       
  • um Arduino Nano microcontrolador Atmel ATmega328.

A montagem do projeto foi esquematizada no Fritzing, assim como indica a Figura 3.
