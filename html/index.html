Sistemas Internet -Sensor de temperatura com Arduino

O sistema desenvolvido tem o intuito de fornecer informações da temperatura de determinado local, em tempo real.


A Figura 1 refere-se ao Arduino utilizado no projeto em questão.



Os componentes eletrônicos usados no projeto foram: 

    
   
  • uma placa de rede Ethernet Shield W5100;
   
  • um sensor de temperatura a prova d’água DS18B20;
   
  • um display LCD 16x2;
   
  • uma protoboard; 
        
  • jumpers;
       
  • um Arduino Nano microcontrolador Atmel ATmega328.
A montagem final está indicada na figura 2.


A montagem do projeto foi esquematizada no Fritzing, assim como indica a Figura 3.
